## OpenTrivia API Library Namespaces
The namespaces available in the *Tudormobile.OpenTrivia* library are described below. In several cases, including the namespace is required to expose extension methods provided.

### Tudormobile.OpenTrivia.dll

```
using Tudormobile.OpenTrivia;
using Tudormobile.OpenTrivia.Extensions;
```

- [Tudormobile.OpenTrivia](Tudormobile.OpenTrivia.yml)
    - Root namespace for the library. Essential classes that comprise the api.
- [Tudormobile.OpenTrivia.Extensions](Tudormobile.OpenTrivia.Extensions.yml)
    - Extensions methods.
