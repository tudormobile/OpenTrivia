# Tudormobile OpenTrivia API
**Tudormobile.OpenTrivia** provides a c# library to access the Open Trivia Database API.  

[Source Code](https://github.com/tudormobile/OpenTrivia) | [Documentation](https://tudormobile.github.io/OpenTrivia/) | [API documentation](https://tudormobile.github.io/OpenTrivia/api/Tudormobile.html)
## Getting Started
### Install the package
```bash
dotnet add package Tudormobile.OpenTrivia
```
### Prerequisites
**NONE**

### Dependencies
Microsoft.Extensions.Logging
Microsoft.Extensions.Http

**Tudormobile.OpenTrivia** is released as open source under the MIT license. Bug reports and contributions can be made at [the GitHub repository](https://github.com/tudormobile/OpenTrivia).
