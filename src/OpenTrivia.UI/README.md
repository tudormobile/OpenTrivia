# Tudormobile OpenTrivia UI Library
**Tudormobile.OpenTrivia.UI** provides a WPF library to build windows desktop applications that access the Open Trivia Database API.  

[Source Code](https://github.com/tudormobile/OpenTrivia) | [Documentation](https://tudormobile.github.io/OpenTrivia/) | [API documentation](https://tudormobile.github.io/OpenTrivia/api/Tudormobile.html)
## Getting Started
### Install the package
```bash
dotnet add package Tudormobile.OpenTrivia.UI
```
### Prerequisites
Tudormobile.OpenTrivia (will be automatically installed as a dependency) requires .NET 10.0 or later.

### Dependencies
CommunityToolkit.Mvvm

**Tudormobile.OpenTrivia.UI** is released as open source under the MIT license. Bug reports and contributions can be made at [the GitHub repository](https://github.com/tudormobile/OpenTrivia).
